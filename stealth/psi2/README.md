# psi2

Vite + React + TypeScript app, powered by [Bun](https://bun.sh/).

## Requirements

- [Bun](https://bun.sh/) `>= 1.2`

## Install

```sh
bun install
```

## Scripts

```sh
bun run dev        # start the dev server (vite)
bun run build      # production build
bun run preview    # preview the production build
bun run typecheck  # type-check with tsc -b
bun test           # run bun's test runner
```
